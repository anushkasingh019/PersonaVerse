# 🎭 PersonaVerse – Identity Echo Interface

## 📌 Overview

PersonaVerse is an AI-powered chatbot built using **Streamlit** and **Google Gemini AI**. It allows users to interact with the same AI through different personalities, making conversations more engaging and unique.

This assignment 2 was developed as part of the **MirAI School of Technology – AI Builder Track** internship assignment.

## ✨ Features

* 🎭 Multiple AI personalities
* 💬 Interactive chat interface
* 🤖 Powered by Google Gemini AI
* ⚙️ Adjustable response length (Short, Medium, Detailed)
* 🗑️ Clear chat functionality
* 🔒 Secure API key management using `.env`
* 🌐 Simple and responsive Streamlit interface

---

## 🛠️ Technologies Used

* Python
* Streamlit
* Google Gemini API (`google-genai`)
* Python Dotenv
